💡 生态双栖洞察：
WorkBuddy 通过 5大IM平台 + 4个扩展入口 = 9大触达渠道，同时覆盖 To C 和 To B 两条线：
• To C：微信客服号 + ClawBot + 小程序 + QQ → 个人用户的"移动AI助手"（轻量+高频+零门槛）
• To B：企微 + 飞书 + 钉钉 + 腾讯乐享 → 企业用户的"AI办公工作台"（管控+协作+知识库）
• 双栖：QQ邮箱 → 个人与企业邮件场景兼顾
• 同一个产品，同一套Skill能力，通过不同IM入口服务不同场景 — 这是WorkBuddy"一鱼多吃"的核心策略

关键启示：每个IM背后是不同的企业生态和办公习惯 — 企微=客户管理，飞书=项目协作，钉钉=考勤审批。WorkBuddy选择"全都要"，但也意味着每个场景都不够深入。深度绑定单一生态（如微软 Teams/Outlook）可能比广撒网更有壁垒。
